# Reflections on the Project

Working through these seven problems was a real eye-opener. I finally moved past the basics and started using "pro" tools like Generics, Interfaces, and OOP in a way that actually makes sense for real-world projects.

I used to think TypeScript was just about extra rules, but now I see it’s about making my code smarter. Using Type Guards and Utility Types felt like having a safety net that caught bugs before I even finished typing. Plus, using Inheritance made it so much easier to keep everything organized and reusable.
